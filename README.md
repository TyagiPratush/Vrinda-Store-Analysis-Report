# 📊 Vrinda Store Data Analysis Dashboard (Excel)

## 📌 Project Overview

This project presents an interactive sales analysis dashboard for **Vrinda Store** using **Microsoft Excel**. The dashboard provides valuable business insights by analyzing sales performance, customer demographics, order status, sales channels, and regional performance for the year **2022**.

The objective of this project is to transform raw sales data into meaningful visualizations that support data-driven business decisions.

## 🎯 Objectives

- Analyze overall sales performance.
- Identify top-performing states.
- Compare sales between men and women.
- Analyze customer age groups.
- Evaluate order status.
- Compare different sales channels.
- Create an interactive dashboard using Pivot Tables and Pivot Charts.

## 📈 Dashboard Features

The dashboard includes the following analyses:

- 📅 Sales vs Orders Trend
- 👨‍👩‍👧 Men vs Women Sales Comparison
- 📦 Order Status Analysis
- 🌍 Top Performing States
- 👥 Age & Gender Analysis
- 🛒 Sales Channel Performance
- 📊 Interactive Excel Dashboard

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning
- Dashboard Design

## 📊 Key Insights

- Women contribute a higher percentage of purchases than men.
- Adult customers form the largest customer segment.
- A few states contribute the majority of total sales.
- Online marketplaces are major contributors to revenue.
- Most orders are successfully delivered with a very small cancellation rate.

## 📁 Dataset

The dataset contains information such as:

- Order ID
- Customer Gender
- Age
- State
- Sales Amount
- Order Status
- Sales Channel
- Product Category
- Order Date

## 🚀 How to Use

1. Download the Excel workbook.
2. Open it in **Microsoft Excel 2019/2021 or Microsoft 365**.
3. Navigate to the Dashboard sheet.
4. Use the slicers to filter data interactively.
5. Explore charts and KPIs for business insights.
